Changes

Visual Changes

  1. Background Image was changed. On longer pages the image from the design plan would be asymmetric

    Page_design2.PNG => bg_light.jpg

    Page_design_dark => bg_dark.jpg

  2. Font type was changed. The last font wouldn't load properly and when it did it wasn't readable.

    font-family:"sendha"; => Arial, Helvetica, sans-serif on all p-tags and h1-tags

    font-family:"sendha"; => 'Courier New', Courier, monospace on all h2-tags to separate out page headers

  3. Sidebar is changed. The color was too easy to overlook and it got a new alternative (Dark/Light mode) since the placement of this function wasn't specified in the design plan.

    Light Mode: color:lightgray; => color:black;

    Dark Mode: not specified => color:rgba(100,0,0,0.9)

    Sidebar Items: History, Favorites => History, Favorites, Dark/Light Mode

  4. Cards layout is changed. The pyramid model for the cards layout was flawed because the cards at the bottom at the pyramid were too small to be read.

    Cards Layout: Pyramid Model => Model of 1 large card, then 2 small cards, repeated.

JavaScript Changes

  1. hover_menu.js, dark_mode.js are now functions within script.js. This is to increase modularity.

  2. fetch_bot.js is changed. APIs weren't allowed, so we replaced this function with the favorite function

    fetch_bot.js => folder: local_favs_scripts, file: genereate_favorite_table

    The folder contains a unique script for each HTML page to store locally from.

  3. search_bar.js is changed. We lost group members unannounced and therefore also capacity. This was changed to history/log function to finish the project faster

    search_bar.js => folder: local_log_scripts, file: genereate_history_table

    The folder contains a unique script for each HTML page to store locally from.

structure change

  1. style.css has changed name and directory position. This is to enhance directory structure and give it a more fitting name

    style.css => stylesheets/main.css

  2. circuit_head.ico has changed name and directory position. This is to enhance directory structure and give it a more fitting name

    circuit_head.ico => images/favicon.png
